# Dove: The Code, Governança Criativa como Diferencial

## O que é
Dove identificou que ferramentas de IA generativa distorciam padrões de beleza. Em parceria com Pinterest, retreinou algoritmos com imagens de mulheres reais. 4 bilhões de impressões globais, 38% dos consumidores com maior confiança na marca.

## Ângulo de uso para a Pira Labs
Argumento de governança e ética como diferencial humano — a decisão de usar IA para corrigir a tecnologia partiu de julgamento estratégico humano. Para agência: delegar julgamento criativo sem curadoria não é eficiência, é risco de cliente.
