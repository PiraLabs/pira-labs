# VML: Have My Agent Call Your Agent, Marca na Era dos Agentes

## O que é
Tese central: o próximo cliente de uma marca não é só uma pessoa, é também um algoritmo. Propõe Brand Agentic Assistant e Brand API. Robots.txt foi criado em 1994 — quem não redesenhar como expõe dados para máquinas vai perder visibilidade antes de perder clientes.

## Ângulo de uso para a Pira Labs
Para agência: quando agente faz a busca, marcas não nomeadas no prompt inicial correm risco de commoditização. A pergunta deixa de ser "como apareço no Google" e vira "o que um agente encontra sobre mim".
