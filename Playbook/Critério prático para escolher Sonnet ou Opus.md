T: Critério prático para escolher Sonnet ou Opus
F: playbook-a-v5.md
TP: framework
FN: Define qual modelo usar conforme complexidade, custo e profundidade exigida. Use antes de iniciar uma tarefa para evitar desperdício de contexto e escolha errada de modelo.
PERSONA: —
ETAPAS:
- Triagem rápida, rascunho inicial: Use Sonnet.
- Trabalho estratégico, decisão importante: Use Opus.
- Tarefa simples, resposta direta: Use Sonnet.
- Sessão longa com muitos documentos: Use Sonnet.
- Regra prática: Use Sonnet por padrão. Mude para Opus quando o problema for complexo e a resposta tiver consequência real.
REGRAS: Desnecessário usar Opus em tarefa simples. Opus consome contexto mais rápido.
SC: Mude para Opus quando o problema for complexo e a resposta tiver consequência real.
OF: —
CP: tipo de tarefa, complexidade do problema, consequência da resposta, volume de documentos na sessão.
OT: Depende da disponibilidade dos modelos Sonnet e Opus.
