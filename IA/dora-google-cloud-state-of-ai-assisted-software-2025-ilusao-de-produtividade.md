# DORA + Google Cloud: State of AI-Assisted Software 2025, a Ilusão de Produtividade

## O que é
Pesquisa com 5.000 profissionais de tecnologia globalmente. Identifica sete capacidades organizacionais que determinam se IA gera ou destrói valor. Dado mais contraintuitivo: desenvolvedores com IA têm 19% menos produção real mas se sentem 20% mais eficientes. 30% têm pouca confiança no código da IA, mas 90% usam.

## Ângulo de uso para a Pira Labs
IA é amplificador, não solução — em organizações frágeis, expõe e acelera problemas existentes. A ilusão de produtividade documentada é argumento poderoso para diagnóstico antes de adoção.
