# MIT Consent in Crisis: Dados Proprietários como Ativo Estratégico

## O que é
Primeiro estudo longitudinal sobre restrições de consentimento em dados de treinamento de LLMs. Mais de 28% das fontes de maior qualidade bloqueadas via robots.txt; 45% do C4 restrito por termos de serviço.

## Ângulo de uso para a Pira Labs
Dados proprietários de empresas viram ativo crescentemente valioso. Para clientes de saúde e jurídico: os dados internos provavelmente não estão sendo tratados como ativo estratégico.
